# アーカイブ

このディレクトリは**廃止・参照用**の格納先です。

## 整理内容（実施済み）

- **バックアップ** (`.backup`)・古い設定 (`old-configs/`)・完了報告の複製は削除済みです。
- 週次レポートやプロトコル文書は **`docs/`** に集約されています。

## 参照先

- プロジェクト構成: ルート **`PROJECT_STRUCTURE.md`**
- ドキュメント一覧: **`docs/INDEX.md`**
- 環境変数: **`docs/ENV_VARIABLES_CHECKLIST.md`**

今後、廃止したコード・設定を残す場合はこのフォルダに置き、必要に応じてこの README を更新してください。
